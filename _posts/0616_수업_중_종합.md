1. [문제 상황]
 내 장바구니에 담긴 물건이 다음과 같습니다. 전체 금액을 계산하여 출력하는 프로그램을 작성해 봅시다.


```python
pay = 0
lst = []
while True:
  name = input("상품명을 입력하세요. : ")
  cost = int(input(f"{name}의 가격을 입력하세요 : "))
  pay = pay + cost
  lst.append([name, cost])
  if input("즐거운 쇼핑을 계속하시겠습니까? (y/n) ") == "n":
    print("\n\n주문 목록\n")
    print(f"{lst}\n총 금액은 {pay}원 입니다.")
    break
  else:
    pass
```

2. 자동계량장비를 사용하는 종량제 시범사업을 실시한 결과, 전년에 비해 음식물쓰레기가 평균 25% 감량됐으며, 이에 따른 식재료 구매 비용 절감액은 한 가구당 연간 19만 원 정도에 달할 것으로 조사되었습니다. 다음 조건에 따라 이 장비를 컨트롤하는 프로그램을 구현해 보시오.


```python
allweight = float(0)
apt = [["101-101", 7788, float(0)], ["101-102", 7788, float(0)], ["101-201", 7788, float(0)], ["101-202", 7788, float(0)], ["101-301", 7788, float(0)], ["101-302", 7788, float(0)]]

while allweight < 20:
  whois = input("동호수 : ")
  ps = input("비밀번호 : ")
  for i in range(len(apt)):
    if apt[i][0] == whois and apt[i][1] == int(ps):
      dmaTm = input("무게 : ")
      allweight = allweight + float(dmaTm)
      apt[i][2] = apt[i][2] + float(dmaTm)
      print("총량 : {0:.1f}".format(allweight))
      print("처리 완료\n")
      break 
else:
  print("한계량 초과!")

for i in apt:
  print(f"{i[0]} : {i[2]}kg")
print("음식물쓰레기 무게(초과됨) : {0:.1f}kg".format(allweight))
```

3. [문제 상황]
 K마트에서는 오전 판매 촉진을 위하여 타임세일을 실시하기로 했습니다. 그러나 준비한 상품이 고객들이 선호는 것이라서 마감 시간 전에 모두 판매될 수도 있습니다. POS 시스템이 이것을 처리하도록 프로그램을 작성해 보시오.


```python
count = 5
while count != 0:
  inp = int(input("time: "))
  if inp >= 1200:
    break
  else:
    cn = int(input("count: "))
    if count - cn < 0:
      print("failure\n")
      continue
    count = count - cn
    print(f"{cn} sale.\nremain: {count}\n")


print(f"\nfinish\nremain products count: {count}")
```
