1. DAUM에서는 주기적으로 비밀번호를 변경하여 개인정보를 안전하게 보호하고 있습니다. ID는 ‘알파벳+숫자’ 의 조합으로, ‘kyunghee8323’과 같이 만들 수 있습니다. 새로 변경하려는 비밀번호에서 연속된 3글자가 ID에 들어 있다면 그것은 사용이 허락되지 않습니다. 위에서 설명한 것과 같이 비밀번호 변경 가능 여부를 체크한 후에 비밀번호를 변경하는 프로그램을 작성하시오. 


```python
id = "kyunghee8323" #1234567890

lu = True
while lu == True:
  ps = input("변경할 비밀번호 입력: ") #456
  for i in range(len(id) - 2): 
    if id[i : i + 3] in ps:
      print("보안상 이 비밀번호를 사용할 수 없습니다.")
      break
  else:
    print("사용할 수 있는 비밀번호입니다.")
    lu = False
```

    변경할 비밀번호 입력: 323
    보안상 이 비밀번호를 사용할 수 없습니다.
    변경할 비밀번호 입력: 323
    보안상 이 비밀번호를 사용할 수 없습니다.
    변경할 비밀번호 입력: 91566
    사용할 수 있는 비밀번호입니다.
    

2. 요트 경주는 물 위에 부표를 띄워 표시해 둔 코스를 정해진 시간 내에 가장 빨리 완주하는 게임입니다. 육상에서 하는 스포츠와는 다르게 바람, 조류, 파도 등 여러 영향을 많이 받기 때문에 레이스를 여러 번 하여 종합점수로 순위를 정합니다. 5팀이 출전하여 3번의 기록이 다음과 표와 같을 때, 각 팀 기록의 평균을 구하여 출력하는 프로그램을 작성해 보시오.


```python
score = [[55, 48, 60], [77, 60, 55], [49, 65, 60], [70, 58, 63], [54, 50, 58]]
num = 0
for n in range(1, len(score) + 1):
  for y in score[n - 1]:
    num = num + y
  print(f"{chr(64 + n)}번 팀의 평균 점수: {round(num/3, 2)}")
  num = 0
```

    A번 팀의 평균 점수: 54.33
    B번 팀의 평균 점수: 64.0
    C번 팀의 평균 점수: 58.0
    D번 팀의 평균 점수: 63.67
    E번 팀의 평균 점수: 54.0
