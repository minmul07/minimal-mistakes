1. lst1에는 [11, 22, 33, 44, 55, 66, 77, 88, 99]가 할당되어 있다. lst2 변수에서도 lst1에 접근할 수 있도록 한 다음, lst2를 이용하여 3번째 요소의 값을 35로 변경한 후 결과와 같이 lst1이 출력되도록 프로그램을 작성하시오.


```python
lst1 = [11, 22, 33, 44, 55, 66, 77, 88, 99]
lst2 = lst1
lst2[2] = 35
print(lst1)
```

    [11, 22, 35, 44, 55, 66, 77, 88, 99]
    

2. flower1에 저장된 ['rose', 'lily', 'cosmos', 'Tulip']을 원본 값 그대로 보존하기 위해 flower2를 새로 만들어 flower1의 값을 복사한 후, flower2의 마지막 값을 ‘Chamomile’로 변경하여 결과와 같이 flower1과 flower2가 출력되도록 프로그램을 작성하시오.


```python
flower1 = ['rose', 'lily', 'cosmos', 'Tulip']
flower2 = flower1.copy()
flower2[-1] = 'Chamomile'
print(flower1)
print(flower2)
```

    ['rose', 'lily', 'cosmos', 'Tulip']
    ['rose', 'lily', 'cosmos', 'Chamomile']
    

3. 현재 lst1에는 [1, 3, 5, 7], lst2에는 [2, 4, 6, 8]의 값이 저장되어 있다. 이 두 리스트를 결합하여 lst3을 만들고 출력 결과와 같이 되도록 프로그램을 작성하시오. 


```python
lst1 = [1, 3, 5, 7]
lst2 = [2, 4, 6 ,8]
lst3 = lst1 + lst2
print(lst3)
```

    [1, 3, 5, 7, 2, 4, 6, 8]

tpl = ('o', 'x', 'o', 'o', 'x')
